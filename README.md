# vimnws : A News Application <br /> 

VimNews is a live News Application.<br />
> Fetch Latest News from The Wall Street Journal.<br />
> User Login And SignUp <br />
> Bookmark And like your Favorite News Headline <br />

>> Technologies: 
 > Flutter,<br />
 > Firebase Authentication,<br />
 > FireStore Database,<br />
 > RESTApi<br />
